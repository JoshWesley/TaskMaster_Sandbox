# Roadmap

## Near Term

- Add incremental Outlook synchronization by folder and timestamp.
- Materialize conversation summaries into work items automatically.
- Persist user ordering, drag-and-drop priorities, and saved dashboard views.
- Expand dashboard widgets for kanban, sender, folder, and domain views.

## Medium Term

- Add rule-based due-date extraction and richer urgency scoring.
- Track customer, supplier, and partner workload trends over time.
- Introduce backlog review workflows such as Today, This Week, Waiting, and Someday.

## Future Extensions

- Optional integrations for Microsoft Graph, Teams, or Planner.
- AI-assisted action extraction once local or approved models are available.
- Background services for faster refresh and scheduled analytics snapshots.
